📋 학생 계정 일괄 생성

CSV 또는 Excel 명렬표를 업로드하면 학생 계정이 한 번에 만들어집니다.
학생들은 학년·반·번호·비밀번호로 바로 로그인할 수 있어요.

1️⃣ 명렬표 양식 다운로드

아래 버튼을 눌러 양식을 받고, Excel에서 학생 정보를 입력하세요.
컬럼 순서: 학년 / 반 / 번호 / 이름 / 비밀번호

2️⃣ 작성된 명렬표 업로드

CSV(.csv) 또는 Excel(.xlsx) 파일을 선택하세요.

3️⃣ 계정 생성 실행

미리보기가 정상이면 아래 버튼을 누르세요. 학생당 약 1초씩 걸립니다.

🔴 라이브 질문 세션

수업 중 "지금, 이걸 보고 질문!"을 열면, 학생 화면에 빨간 배너가 떠요. 학생이 보낸 질문이 아래에 실시간으로 모입니다.

📚 수업 자료
